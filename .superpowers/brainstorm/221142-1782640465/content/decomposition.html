Spec 1 — Tab/Header 集成重写
保留 Dockview v7，完全重写 Tab/Header 集成层。去掉 CSS hack，改用 Dockview 原生 API 控制样式。重写 ReactContentRenderer / ReactTabRenderer 的 Context 桥接逻辑。
文件: DockviewContainer.tsx, TabHeader.tsx, useTabManager.ts, index.css
Spec 2 — 共享代码还原 + CWD
还原 8 个共享文件的改动到 master。新增 web 专有 CWD REST API（GET/PUT /api/sessions/{chatID}/cwd）。前端 CwdProvider 改为每次切换会话从后端拉取 CWD。
文件: serverapp/*, agent/*, channel/web/web_api.go, CwdProvider.tsx
Spec 3 — 流式数据模型 + Store 重写
重写 ProgressStore 和 useProgressStream。实现 stream-only patch（非替换）、carry-forward、迭代快照、去重保护。定义 Spec 4/5 共用的数据类型和 Store 接口。
文件: useProgressStream.ts, progressStore.ts, types/shared.ts
Spec 4 — 渲染 + 3 级折叠
重写 AssistantMessage / ProgressPanel / IterationHistory。实现 3 级折叠（all/minimal/none）、连续工具合并卡片、无边框折叠行（▸/▾）。流式实时渲染。
文件: components/agent/*, useCollapseLevel.ts
Spec 5 — 会话生命周期显示
新会话打开、历史压缩后恢复、会话切换时的状态恢复。重写 useChatMessages 历史水合逻辑、HistoryCompacted 处理、session 切换重置。
文件: useChatMessages.ts, AgentPanel.tsx, wsConnection.ts
